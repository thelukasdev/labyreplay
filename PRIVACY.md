# Privacy

LabyReplay is designed as a local-only highlight recorder.

- No data is uploaded by the addon.
- No analytics or tracking are used.
- Highlight files are written to the user's local file system only.
- Clipboard usage only happens when the user explicitly requests a share image.
- The addon does not read passwords, session tokens, account credentials, or unrelated user files.

If server-specific integrations are added later, they must be documented here before release.
